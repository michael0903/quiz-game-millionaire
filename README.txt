=====================================================
   WHO WANTS TO BE A MILLIONAIRE? - QUIZ GAME
=====================================================

QUICK START
-----------
1. Double-click "play-game.bat" to start the game
2. Or open "index.html" in your web browser

WHAT'S INCLUDED
---------------
- Complete quiz game with 15 questions
- Multiple difficulty levels with increasing prize money
- Three lifelines: 50/50, Phone a Friend, Ask the Audience
- Background music and sound effects
- Bilingual support (English/Traditional Chinese)
- Fully self-contained - no internet connection required

SYSTEM REQUIREMENTS
-------------------
- Windows 7 or later (for batch launcher)
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

HOW TO PLAY
-----------
1. Choose your language (English or Traditional Chinese)
2. Answer 15 multiple-choice questions correctly
3. Use lifelines strategically to help with difficult questions
4. Reach the million dollar prize!

LIFELINES
---------
- 50/50: Eliminates two wrong answers
- Phone a Friend: Get a hint from a virtual friend
- Ask the Audience: See what the audience thinks

PRIZE STRUCTURE
---------------
Question 1-5:   $100 - $1,000
Question 6-10:  $2,000 - $32,000  (Safe Haven at $1,000)
Question 11-15: $64,000 - $1,000,000  (Safe Haven at $32,000)

TECHNICAL DETAILS
-----------------
- Built with HTML5, CSS3, and JavaScript
- No external dependencies
- Works offline
- Responsive design optimized for desktop

TROUBLESHOOTING
---------------
If the game doesn't start:
1. Try opening index.html directly in your browser
2. Make sure JavaScript is enabled
3. Check that all files are in the same folder
4. Try a different web browser

SUPPORT
-------
This is a standalone game with no online support.
All game files are included in this package.

Enjoy the game!
